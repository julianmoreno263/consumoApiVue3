# Ejemplo curso Udemy Vue 3

- [Preview page](https://consumoapivue.netlify.app/)

Este es un ejemplo del curso de Udemy de vue 3, trabajando con componentes y le composition api, consumimos
la api de jsonplaceholder.


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
